# Rasperry Pi 5 VGA666 240P MAME

### 1 - Prepare SD card via rpi-image
Install the latest Raspberry Pi OS 64-Bit on the SD card.
```bash
sudo rpi-imager
```

### 2 - Update the whole system
```bash
sudo apt update && sudo apt upgrade -y
```
